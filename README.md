## PixelFight 게임 설명서
EN: <https://github.com/Usual24/PixelFight/blob/main/README-EN.md>
PixelFight는 2D 픽셀 스타일의 1:1 대전 게임입니다. 당신의 목표는 고대 신화 속 존재, "픽셀"들 중 하나를 선택하여 상대방의 체력(HP)을 0으로 만들어 승리하는 것 입니다.

---

### 게임 시작 및 조작 방법

#### 게임 시작
1.  **픽셀 선택**: Player 1은 A/D 키, Player 2는 왼쪽/오른쪽 화살표 키로 선택합니다.
2.  **게임 시작**: 선택 후 Enter 키를 누릅니다.
3.  **자동 플레이**: 게임 중 Enter 키를 다시 누르면 가상의 픽셀과의 대전 모드가 활성화됩니다.

#### 조작 방법

| 조작 | Player 1 (왼쪽) | Player 2 (오른쪽) | 비고 |
| :--- | :--- | :--- | :--- |
| **이동** | A (좌), D (우) | ← (좌), → (우) | |
| **점프** | W | ↑ | |
| **다운** | S | ↓ | |
| **대시** | A 또는 D를 빠르게 두 번 입력 | ← 또는 →를 빠르게 두 번 입력 | **더블탭**으로 발동 |
| **스킬 I** | F | 1 | |
| **스킬 II** | G | 2 | |
| **스킬 III** | H | 3 | |

### 픽셀 소개

| 픽셀 | 특징 | 스킬 I | 스킬 II | 스킬 III | 팁 및 특수 효과 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **불의 화신(Red)** | 화염 기반 공격, 연계 콤보 | **신성한 불꽃** (크기/위력 증가) | **작열하는 태양** (공중 폭발) | **천상의 유성** (9연속 콤보) | **화상** (초당 1 피해) 유발. |
| **빙하의 영혼(Blue)** | 방어/반격, 얼음 속성 | **서릿발 원반** (**3회 사용 시 강화**) | **부유의 축복** (240프레임 무중력) | **빙하의 영역** (범위 얼림) | **얼음** (이동 속도 60% 감소) 유발. |
| **질풍의 정령(Mint)** | 고기동성, 바람 속성 | **돌풍의 칼날** (**20회 충전 시 강화**) | **칼날 폭풍우** | **천둥 소환** (태풍) | |
| **대지의 거신(Brown)** | 탱커, 돌진/방어 | **대지 관통창** (**길게 눌러 차지 가능**) | **바위 장벽 생성** | **진격의 쇄도** (돌진 파쇄) | 스킬 I **차지 시 위력 대폭 증가**. |
| **자연의 수호자(Green)** | 회복 & 함정 설치 | **생명의 꽃가루** (랜덤 효과) | **포획의 덩굴** | **원시의 씨앗** (자동 발아) | |
| **태양의 사자(Yellow)** | 광속 공격, 투사체 반사 | **빛의 검** (2스택 시 레이저) | **광휘의 방패** (**투사체 반사**) | **태양의 대검** (480프레임 지속) | |
| **어둠의 지배자(Gray)** | 흡혈 & 그림자 마법 | **영혼 추적자** | **무적의 장막** (**HP 10% 이하 시 자동 활성화**) | **어둠의 의식** (지속 피해 + 흡혈) | **무적** (스킬 II 활성화 시) 상태. |

---

### 게임 진행 및 승리
* **승리 조건**: 상대방 HP를 0으로 만듭니다.
* **특수 효과**: 화상, 얼음, 기절(입력 불가), 무적 등이 존재합니다.
* **재시작**: 게임 종료시 Enter 키로 재시작 가능합니다.
